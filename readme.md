Requirements

Node and npm
------------

Installation:

Build
=====

Install dependencies
--------------------
```shell
npm install
```

start webserver
---------------
```shell
gulp server
```

-> localhost:8080


one time build
-------------
```shell
gulp 6to5
```


watch files for changes -> trigger build
----------------------------------------
```shell
gulp watch
```
