Requirements
============

Node and npm
------------

Installation:
https://gist.github.com/isaacs/579814



Build
=====

Install dependencies
--------------------
npm install


one time build
-------------
gulp 6to5


start webserver
---------------
gulp server

-> localhost:8080


watch files for changes -> trigger build
----------------------------------------
gulp watch



Atom Text Editor
================

Install Refactoring
-------------------

Open Preference -> Install

Search for refactor -> install
  - refactor
  - js-refactor

Now you can refactor with ctrl-alt r
